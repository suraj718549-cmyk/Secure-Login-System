# 🔐 Secure Login System

A modern cybersecurity-themed secure authentication web application built using Python and Flask. This project demonstrates secure login practices including password hashing, CSRF protection, session management, SQL injection prevention, and futuristic cyberpunk UI design.

---

## 🚀 Features

* 🔑 Secure User Registration & Login
* 🛡️ Password Hashing using bcrypt
* 🔒 CSRF Protection
* 🧠 SQL Injection Prevention
* 👤 Session Management & Logout
* 🔄 Password Reset Functionality
* 📱 Responsive Cyberpunk UI
* ⚡ Real-Time Authentication Portal
* 🔐 Optional Two-Factor Authentication (2FA)
* 📊 Admin Dashboard Support
* 🚨 Failed Login Protection
* 🌌 Futuristic SOC-Level Design

---

## ▶️ How to Run

```

### 2. Open Project Folder

cd Secure-Login-System
```

### 3. Install Requirements

pip install -r requirements.txt
```

### 4. Run Application

python run.py
```

### 5. Open Browser
